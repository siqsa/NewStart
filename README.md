# ReflectNote

## 소개

ReflectNote라는 이름은 '생각할 수 있는 일기장'을 만들고자 하는 취지에서 지었습니다. 사용자가 일기를 쓰고 지정한 시간에 맞춰 일기 내용이 푸쉬 알림으로 핸드폰에 뜨게 됩니다. 전날에 적은 일기 내용을 다시금 떠올리며 실수는 반성하고, 기쁨을 다시금 느낄 수 있는 앱이 되고자 합니다.
